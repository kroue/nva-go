## About

**NVAGo** is a comprehensive **online ordering** and **Point-of-Sale (POS) system** specifically designed for **NVA Printing Services.** This capstone project modernizes traditional printing service operations by integrating:

- Buy-Online-Pickup-In-Store (BOPIS) functionality
- Real-time customer-designer communication
- Cashless payment integration (GCash)
- Streamlined order management and tracking

The system addresses operational challenges such as long wait times, communication barriers, and limited digital payment options while enhancing customer satisfaction and business efficiency.

## Key Features

### For Customers (Mobile Application)
- **Online Ordering** - Place orders anytime, anywhere via mobile app
- **Direct Communication** - Chat with graphic designers for design collaboration
- **Cashless Payments** - Secure GCash integration with QR code scanning
- **Order Tracking** - Real-time status updates and notifications
- **Design Approval** - Review and approve designs before printing
- **Order History** - View past orders for easy reordering
- **Digital Receipts** - Download receipts after transactions

### For Employees (NVA printing services Staff)
- **Order Management**- Process and update customer orders
- **Payment Validation** - Verify and confirm customer payments
- **Design Upload**- Send design previews for customer approval
- **Customer Support** - Respond to customer inquiries via in-app chat
- **Status Updates** - Manage order progress from start to finish

### For Admin (Developers)
- **User Management** - Add, edit, and remove user accounts
- **System Monitoring** - Track system activity and generate reports
- **Service Configuration** - Update products, services, and pricing
- **Security Control** - Manage access and maintain system integrity
